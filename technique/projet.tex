\section{Diagramme des projets}

Nous présentons ici la gestion des projets.\\
\begin{itemize}
\item Projet : Représente un objet contenant les données d'un
projet. Après sa création, il est stocké dans ProjetManager.\\
\item ProjetManager : C'est l'objet qui gère les projets, il
s'occupe de leur création, recherche.\\
\item TeachingData : Il gére le stockage des projets dans
leur intégralité. Le ProjetManager lui fait appel pour toutes les
opérations.\\
\item Trash : C'est l'objet qui collecte tous les
projets mis à la poubelle par leur créateurs. Il les conserve
jusqu'a leur suppression définitive.\\
\item TeachingManager : Il gère l'ensemble des projets pour la création ou la suppression.\\
\end{itemize}
