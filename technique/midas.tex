\section{Rapport avec le projet}

Midas va nous permettre d'éditer des documents en ligne à partir du navigateur web de l'utilisateur.

\section{Qu'est ce que c'est ?}

C'est un éditeur de fichier texte implanté de base dans mozilla.

\section{Avantage et désavantage}

Les avantages sont :
\begin{itemize}
\item fourni de base dans mozilla.
\item nombreuses méthodes.
\item simple d'emploi.
\end{itemize}

Les désavantages sont :
\begin{itemize}
\item on ne le connait pas.
\end{itemize}
